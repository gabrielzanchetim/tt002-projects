# Car Management System

Este projeto é um sistema básico para gerenciamento de carros, utilizando Python e arquivos de texto para armazenamento.

## Funcionalidades

- Adicionar novos carros ao banco de dados local.
- Carregar e listar informações de carros armazenados.
- Verificar se uma placa já está registrada no sistema.
- Gerar placas aleatórias para testes.

## Como Executar

1. Certifique-se de que o arquivo `projeto1.py` e o arquivo `carros.txt` (se existente) estejam no mesmo diretório.
2. Execute o arquivo `projeto1.py` com Python:
   ```bash
   python projeto1.py
   ```
